VDF Verify Calculator (from: mmx-node-notes)

Simple calculator to estimate max timelord speed an mmx-node config can handle.

Read 3 values/properties from mmx-node:

  • Network Speed: Value shown in WebGUI / NODE section, as VDF Speed.
  • Look at WebGUI / NODE / LOG / TERMINAL entries:
  • Verified VDF for height x, delta = x.x sec, took x.x sec (with reward)
  • Verify VDF Time: Value 'took x.x sec' in line above.
  • With Reward: If '(with reward)' in line above, check box. If not, uncheck.

NOTE: Network speed should be stable, WebGUI / NODE / VDF SPEED.
NOTE: Verified VDF delta in line above should be at ~10 sec (+/- 0.5).
